CSS Overriding Variables

Override Global Variable With Local Variable

Global variables can be accessed/used through the entire document, while local variables can be used only inside the selector where it is declares.
Sometimes we want the variables to change only in a specific section of the page.
Assume we want a different color of blue for buttons elements. Then, we can re-declare the variable inside the button selector. WHen we use var inside this selector, it will use the local variable value declared.
We see the local variable will override the global variable for the button elements, for example:

:root { --blue: blue; --white: white; } body { background-color: var(--blue); } h2 { border-bottom: 2px slid var(--blue); } .container { color: var(--blue); background-color: var(--white); padding: 15px; } button { --blue: #0000ff; /* local variable will override global variable */ background-color: var(--white); color: var(--blue); border: 1px solid var(--blue); padding: 5px; }
