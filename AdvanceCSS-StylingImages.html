Image Filters

The CSS 'filter' property adds visual effects (like blur and saturation) to an element.
Note: The filter property is not supported in Internet Explorer or Edge 12. to change the color of all images to black and white (100% gray): 


            img { 
                filter: grayscale(100%);
                }
