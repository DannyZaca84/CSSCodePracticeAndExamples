CSS 2D Transform

CSS transforms allow you to move, rotate, scale, and skew elements. This practice is focused on the CSS property

CSS 2D Transforms Methods

With the CSS transform property you can use the following 2D transformation methods:

  • translate()
  • rotate()
  • scaleX()
  • scaleY()
  • scale()
  • skewX()
  • skewY()
  • skew()
  • matrix()
